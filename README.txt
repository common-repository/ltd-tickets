London Theatre Direct Plugin quickly and easily turns your WordPress website revenue generating, ticket selling website.

== Description ==

Earn commission by turning your WordPress site into a London Theatre tickets website, powered by London Theatre Direct.

The plugin allows you to quickly import products, venues and categories from London Theatre Direct, creates product archives, category pages, product pages and venue pages, and generate revenue from every sale made via your website. 
Synchronisation functionality automatically keep your site up to date with the latest events, prices and special offers without you having to lift a finger.

The features include:

* Import only the products you want to sell
* Simple design options to ensure the plugin matches your existing website
* One click 'Import All' turns your site into a ticketing website in a single click
* Shortcode builder allows you to add book buttons to your existing pages and posts
* Flexible post types and taxonomies let you retrofit the plugin to your existing ticketing website
* The Data Manager can be configured to automatically import new products, venues and categories
* Automatic synchronisation to ensure prices, special offers and important product info is always up to date
* Become a London Theatre Direct affiliate and earn commission from sales you refer
* Works for all partner types: AWIN (previously Affiliate Window), London Theatre Direct API, and London Theatre Direct Whitelabels
* Clean, minimal, responsive templates that can be simply replaced by your own theme files

For instructions, documentation and tutorials check out the [Plugin Website](https://wordpress.londontheatredirect.com).


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `ltd-tickets` plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on the 'Theatre Tickets' admin page
4. Choose your partner type (AWIN, API, Whitelabel or None)
5. On the 'Import Settings' tab, either choose what you'd like to import, or import everything with a single click

== Frequently Asked Questions ==

= Do I need to be a London Theatre Direct partner to use the plugin? =

No, you don't have to be a partner, but it is recommended. Becoming a London Theatre Direct partner is easy, free, and allows you to earn revenue from the plugin.


= How do I sign up to be a London Theatre Direct partner? =

There are a few ways - feel free to pick the one that best suits you:

1. [Get your own free London Theatre Direct whitelabel](https://partners.londontheatredirect.com/#solution-whitelabel)
2. [Sign up for the London Theatre Direct API](https://iodocs.londontheatredirect.com/member/register)
3. [Join the AWIN program](http://www.awin1.com/cread.php?awinmid=3&awinaffid=140777&clickref=&p=https%3A%2F%2Fui.awin.com%2Fmerchant-profile%2F610)


= Which partner programme has the best benefits? =

You will earn revenue from all the partner programmes, but signing up as an API user unlocks several additional options in the plugin.
